# Acara 3 Frendy

## Deskripsi
Proyek ini adalah aplikasi web sederhana yang menggunakan HTML, CSS, dan JavaScript untuk melakukan berbagai operasi matematika, menampilkan objek dan data JSON, serta memanipulasi tanggal dan data menggunakan JavaScript. Aplikasi ini juga mengilustrasikan penggunaan berbagai konsep dasar JavaScript seperti variabel, tipe data, fungsi, objek, JSON, perulangan, dan pengambilan data dengan `fetch`.

## Tujuan
1. **Matematika Sederhana:** Menghitung penjumlahan, pengurangan, perkalian, dan pembagian dari dua angka yang dimasukkan oleh pengguna.
2. **Objek dan JSON:** Menampilkan objek sederhana dan data dalam format JSON.
3. **Tanggal:** Menampilkan tanggal dan waktu saat ini.
4. **Perulangan:** Menampilkan hasil perulangan menggunakan `for` loop.
5. **Pengambilan Data:** Mengambil dan menampilkan data dari API menggunakan `fetch`.

## Fitur
- Input angka untuk operasi matematika
- Menampilkan hasil operasi matematika
- Menampilkan informasi dari objek JavaScript
- Menampilkan data JSON
- Menampilkan tanggal dan waktu saat ini
- Menampilkan hasil perulangan
- Mengambil dan menampilkan data dari API
